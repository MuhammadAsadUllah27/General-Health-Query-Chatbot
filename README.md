# General-Health-Query-Chatbot
In this project we made a chatbot that is used for the general health queries. 
## Task Objective
Sometimes there is no doctor present at time and patient feel a little uncomfortable. At this time a General Health Query chatbot may give advise according to the symptom that the condition is serious or not. If the condition is serious then go to the doctor immediatly. In this way a General Health Query chatbot help us
## Models Applied 
In this project I used 'gemini-2.5-flash' a model ofGoogle AI Studio.
## Key Results and Findings
The results of this projects are good and General Health Query Chatbot is doing well. I applied fillter in this project so that it will not give the harmfull advise
